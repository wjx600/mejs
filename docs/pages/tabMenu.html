说明

为页面和功能提供导航的菜单列表。

导航菜单是一个网站的灵魂，用户依赖导航在各个页面中进行跳转。

一般分为顶部导航和侧边导航，顶部导航提供全局性的类目和功能，侧边导航提供多级结构来收纳和排列网站架构。

HTML

导航菜单约定html结构，样式可以根据实际情况用css样式定义。

<div id="cate"> <!--被渲染菜单id，根节点-->
  <div>
    <div>  <!--菜单标题，可以自定义加任何icon，文字，标签等元素。-->
        <h2>理财</h2>
        <i class="me-ion-o-arrow-right"></i>
    </div>
  </div>
  <div>
    <div> <!--菜单显示内容，可以自定义加任何icon，文字，标签等元素。-->
      <p style="text-align: center;padding-top:20px;">理财</p>
      <img src="https://img30.360buyimg.com/jr_image/jfs/t4243/78/140755024/37239/ccf5d5c1/58afe47aN307e483a.png" alt="" />
    </div>
  </div>
</div>

JS

需要通过me.require导入menu组件js、css文件。具体见me.require用法

me.require(['tabMenu&css'], 'tabMenu'); //加载tabMenu.js、tabMenu.css文件
me.require.ready('tabMenu',function(){
  me.tabMenu.render('cate', {
    leftWidth: '160',
    leftHeight: '200',
    rightWidth: '530',
    rightHeight: '348',
  });
});

API

名称 说明
第一个参数
需要渲染导航元素id		 
me.tabMenu.render('cate');
mode
菜单类型，现在支持水平
string: 'horizontal'
 
me.tabMenu.render('cate', {
  mode: 'horizontal',
  leftWidth: '160',
  leftHeight: '200',
  rightWidth: '530',
  rightHeight: '348',
});
leftWidth
设置菜单标题宽度		 
me.tabMenu.render('cate', {
  leftWidth: '160',
  leftHeight: '200',
  rightWidth: '530',
  rightHeight: '348',
});
leftHeight
设置菜单标题高度		 
me.tabMenu.render('cate', {
