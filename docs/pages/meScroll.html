说明

对系统默认滚动条的优化

使用是请用 me.require 方法按需将meScroll.js加载到页面

调用方式:
me.require(['meScroll'], 'meScroll'); //加载arousel.js文件
me.require.ready('meScroll',function(){   //关于require的使用请查阅对应的文档
	obj.meScroll({
	  cursorcolor:"#ccc",
	  autohidemode:false,
	  .....
	});
})

	autohidemode:false,

	cursorwidth:"10px",
	cursorborder:"1px solid #feeff5",
	railpadding :{right:1},
	cursorborderradius:"3px"
});

API

属性 说明
cursorcolor 设置滚动条颜色,默认为“＃000000”
cursoropacitymin 设置滚动条颜色透明度,范围从1到0，默认为0（隐藏），如果设置了autohidemode为false则无效
cursoropacitymax 设置滚动条激活状态的颜色透明度,范围为1到0，默认值为1（完全不透明度）
autohidemode 滚动条非激活状态隐藏，默认值true，设置为false则一直显示
cursorwidth 设置滚动条宽度，默认5px
cursorborder 设置滚动条边框样式,默认“1px solid #fff”
cursorborderradius 设置滚动条圆角，默认“4px”
background 滚动条滑槽背景颜色，默认为空
railpadding 滚动条滑槽内边距，默认为{top：0，right：0，left：0，bottom：0}
