说明

dialog，mejs弹出框插件，使用时需要require导入，例：

me.require(['dialog'],function(){
  /*
    me.dialog.showDialog(options);
  */
});

方法

方法名 介绍
showDialog 调用弹出框的显示方法
DialogClose 调用弹出框的关闭方法

参数

名称 类型 可选 默认值 描述
url/text string 否 加载的内容URL或者文本
simple boolean 是 false 是否采用精简版本
title string 是 标题，仅在simple为true时可传
close boolean 是 false 关闭箭头，仅在simple为true时可传
OtherClose boolean 否 false 是否允许点击弹框其他区域关闭
width string 是 弹框宽度，仅在simple为true时可传
showPoint string 是 弹窗以某个元素为参考点显示
showPot string 是 弹框的显示位置
callback function 是 弹框的回调函数

代码 

//显示弹出框
me.dialog.showDialog({
  "url":"test.html #id",
  "simple":true,
  "OtherClose":true,
  "title":"hello world",
  "width":"100px"
  },function(){
      //显示后操作
})
//关闭弹出框
me.Dialog.DialogClose(function(){
  //关闭后操作
})
